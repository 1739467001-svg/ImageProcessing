## 1. 🔴 分类 Classification（Week7 主体，最大盲区，计算题高发）

> 之前我把这块当"W8"且只写了"取中点"，其实它是 Week7 的**核心**，且有标准计算题型。

### 1.1 判别函数与决策边界
- 每类有一个**判别函数(discriminant) $d_i(\mathbf{x})$**；判给使 $d_i$ 最大的类。
- 两类**决策边界**：$d_{ij}(\mathbf{x})=d_i(\mathbf{x})-d_j(\mathbf{x})=0$。
- 🔴 **手画线性分类器**（Week7 p11-13 例题）：给一条边界线如 $\frac{3}{7}x+y-3=0$，代入样本点→结果**正/负**判类。例：(2,2)→负→类2；(3,3)→正→类1。**这就是 Quiz8/模拟卷"决策边界"题的真正考法**。

### 1.2 🔴 最小距离分类器 (Minimum-Distance Classifier)
- 每类的**均值向量**：$\mathbf{m}_j=\dfrac{1}{N_j}\sum_{\mathbf{x}\in C_j}\mathbf{x}$。
- 把样本判给**欧氏距离最近**的类均值：$D_j=\lVert\mathbf{x}-\mathbf{m}_j\rVert$，取最小。
- 等价判别函数：$d_j(\mathbf{x})=\mathbf{x}^T\mathbf{m}_j-\tfrac{1}{2}\mathbf{m}_j^T\mathbf{m}_j$，取最大。
- 两类边界线（例 Iris，m₁=[4.3,1.3], m₂=[1.5,0.3]）：$2.8x_1+1.0x_2-8.9=0$。
- **典型计算题**：给两类若干样本 → 求类均值 → 写决策边界 → 判新样本。务必会算。

### 1.3 其它分类器（🟡概念）
- **kNN (k 近邻)**：看新点最近的 k 个邻居投票，多数类胜（例 10-NN）。需选 k、需归一化特征。
- **决策树 Decision Tree**：节点按特征分裂、叶节点定类；参数 MinLeaf(叶最小样本)、树深；易**过拟合**→**剪枝(pruning)**。
- **随机森林 Random Forest**：多棵树投票，降过拟合。
- **SVM 支持向量机**：找**最大间隔(margin)** 超平面，由**支持向量**决定；核(多项式/高斯/tanh)处理非线性。
- **感知机/神经网络**：$d(\mathbf{x})=\sum w_i x_i+w_{n+1}$，过激活函数；多层前馈+反向传播+梯度下降训练；**梯度消失(vanishing gradient)** 是深网难题。
- **交叉验证 (k-fold, 如10-fold)**：划分训练/验证集评估泛化。

## 2. 🟡 CNN 卷积神经网络（Week7 末段，补全）

之前 W7 只有 Conv2D+ReLU，补：
- **激活函数**：sigmoid $\frac{1}{1+e^{-x}}$、tanh、**ReLU=max(0,x)**、softmax(输出层归一为概率)。
- **权重共享 (weight sharing)**：同一卷积核滑过整图，大幅减参数。
- **多通道 3D 核**：5×5×3 核 + 1 bias = **76 个权重**；3 通道相乘求和→1 个输出层。
- **输出尺寸**：valid 卷积每边缩 (F−1)/2，如 5×5 核作用 32×32 → **28×28**。通式 $\lfloor\frac{W-F+2P}{S}\rfloor+1$（讲义没给通式，只给特例）。
- **池化 Pooling**(Week7 p62-63)：MaxPooling 降采样、增强平移不变性。
- **网络结构**：VGG-19 = (Conv+ReLU)×n → MaxPool → … → 全连接FC → softmax。
- **dropout**(随机丢神经元防过拟合)、**数据增强**、**目标检测 R-CNN / YOLO**（了解）。

## 3. 🔴 互相关与模板匹配 + SIFT（week8，全缺）

### 3.1 互相关 Cross-Correlation（模板匹配）
- 模板 g 滑过图 f 求积之和（**不翻核**，与卷积区别）。
- 🔴 **归一化互相关 (Normalised Cross-Correlation)**：
  $$c(i,j)=\frac{\sum\sum f\,g}{\sqrt{\sum\sum f^2\cdot\sum\sum g^2}}$$
  归一化是为**消除局部亮度差异**的影响。廉价替代：先各减均值(zero-mean)。
- **不变性**(重要对比点)：互相关**平移不变(shift-invariant，峰随目标移动)**，但**对旋转、尺度、仿射敏感(NOT invariant)**。
- 用**多个滤波器加权叠加** $H=a_1H_1+a_2H_2+a_3H_3$ 应对旋转/尺度，但叠太多会"糊"。
- 可在**傅里叶域**做：$f\star g=\mathcal{F}^{-1}(F^*G)$（匹配滤波器用复共轭）。

### 3.2 🟡 SIFT (Scale-Invariant Feature Transform)
- 用 **DoG 多尺度**找**关键点(keypoints)**；每个关键点取 **128 维特征向量**。
- 给每个关键点定**主方向**并对齐→实现**旋转不变**；对 **旋转/尺度/仿射都不变**（与互相关相反，这是 SIFT 的优势）。
- 应用：物体识别、**图像拼接(mosaic)**（配 RANSAC + imwarp）。

## 4. 🔴 图像配准与物体计数（Week9，全缺）

### 4.1 物体计数与标记 (Object Counting & Labelling)
- 完整流程：去噪(高斯/中值) → **光照校正**(形态学腐蚀估背景再相减) → **对比拉伸** → **迭代阈值**(T=½(μ₁+μ₂)) → **连通域(4/8连通)** → **连通域标记(labelling + 等价表)** → 数不同标签数 = 物体数。
- （这串正好把你已会的 W3/W4 工具串起来，理解流程即可。）

### 4.2 🔴 图像配准 (Image Registration)
- 用**仿射/单应矩阵**把一张图对齐到另一张。
- 变换矩阵表(平移/缩放/错切/旋转)；**控制点法(control points)** 选对应点求变换。
- 迭代法：变换→**双线性插值**重采样→算**相似度 metric(均方误差 MSE)**→**梯度下降**优化参数→循环。

### 4.3 🟡 RANSAC
- 鲁棒估计**单应矩阵 H**（$\mathbf{x}'=H\mathbf{x}$，3×3 投影矩阵）。
- 反复**随机取最小子集**算 H → 数有多少点符合(内点 inliers) → 留内点最多的模型。能抗大量错误匹配(外点)。

### 4.4 ⚪ 其它
- **fiducial markers**(基准标记)：人为放的标记(如四角黑圆点)做精确定位——**这正是实验项目用的方法**。
- **Nao 机器人地平面**：投影矩阵把图像坐标→真实世界，假设物体在地面("找脚不找头")。
- **SegNet**：编码-解码(auto-encoder)结构做**语义分割**；像 VGG；需 GPU/CUDA。

## 5. 🔴 傅里叶描述子 Fourier Descriptors（Week6 p81-97，全缺）

- **边界 → 复数序列**：沿物体边界取 N 点，每点写成复数 $s(k)=x(k)+j\,y(k)$（x=实轴，y=虚轴）。→ **二维边界降成一维特征**。
- 对 s(k) 做 **DFT(离散傅里叶变换)** 得系数。
- 🔴 核心结论：**低频系数→物体粗略形状/大小；高频系数→细节**。**丢掉高频 = 平滑边界 = 低通滤波**，对小形变/噪声**鲁棒**。
- 取**有限 M 个低频系数**作低维**形状描述子**→快速形状匹配/识别，可做到"类内容差、类间可区分"。
- 不变性思想（标准做法，讲义未给完整公式）：丢 0 号系数去平移、除以 |F(1)| 归一化尺度、取模去旋转。

## 6. 🟡 图像增强点变换菜单（ImageProcessing3，缺）

quiz 只考了对比拉伸，但讲义讲了整套点变换 $s=T(r)$（可能出"给变换式问它干什么"）：
- **负片 (negative)**：$s=L-1-r$（或归一化 $s=1-r$）。翻转明暗。
- **对数变换 (log)**：$s=c\log(1+r)$，c=1，需归一[0,1]。**压缩高动态范围**(如显示傅里叶频谱)、提亮暗部。
- **幂律/Gamma**：$s=c\,r^{\gamma}$。**γ<1 提亮暗部，γ>1 压暗**；**Gamma校正** $s=r^{1/2.5}=r^{0.4}$。
- **分段线性 / 灰度切片 (grey-level slicing)**：突出某灰度区间。
- **位平面切片 (bit-plane slicing)**：把图按 bit 拆成 8 个平面，高位平面含主要信息。

### 🔴 直方图均衡 Histogram Equalisation（讲义给了完整公式，可能出计算题）
- 用**累积分布(CDF)** 做映射：
  $$s_k=T(r_k)=\sum_{j=0}^{k}p_r(r_j)=\sum_{j=0}^{k}\frac{n_j}{n}\quad(\text{再}\times(L-1)\text{取整})$$
  （rk=输入灰度，nj=该灰度像素数，n=总像素，sk=输出灰度）
- **方法**：①统计各灰度频数 → ②算概率 pj=nj/n → ③累加得 CDF → ④×(L−1) 四舍五入 → 得映射表。
- 作用：拉开拥挤的灰度、增强对比。**缺点**：可能放大平坦区噪声、减少灰阶丢细节。改进：**CLAHE(限制对比度自适应均衡)**。

## 7. 🟡 颜色模型与分割方法（IP3，缺）

### 颜色模型
- 🔴 **HSV/HSI**（颜色分割主力）：H 色相、S 饱和度、V/I 明度。$I=\frac{R+G+B}{3}$，$S=1-\frac{3\min(R,G,B)}{R+G+B}$，H 用 arccos 式。
  - **关键应用**：选某颜色物体时**对 H、S 设阈值、忽略 V**→不受亮度变化影响（比在 RGB 上设 6 个阈值稳健）。
- ⚪ **CMYK**(印刷减色)：$[C,M,Y]=[1,1,1]-[R,G,B]$，K=min(C,M,Y)。
- ⚪ **RG 色度**：$r=\frac{R}{R+G+B}$ 等，去亮度。 **Lab**：感知均匀。 **灰度**：$(R+G+B)/3$。

### 分割方法（quiz 只考迭代阈值，讲义还有）
- ⚪ **P-tiles**：已知物体占 P% 面积→取阈值使 P% 像素入选。
- 🟡 **双阈值 / 分水岭 (Watershed)**：能**分割接触/重叠的物体**（单阈值做不到）——常考概念点。
- ⚪ **四叉树 (Quadtree) split-merge**：按谓词(方差/熵)递归分裂合并。
- 🟡 **运动分割/背景相减**：$|I_{bk}-I|>A$(A≈30)；背景更新 $I_{bk}^{new}=(1-\alpha)I_{bk}^{old}+\alpha I_{new}$（α 越大更新越快）。

## 8. 🟡 形态学高级算子（IP4，缺）

quiz 只考了腐蚀/膨胀，讲义还有：
- 定义本质：**膨胀=邻域取 max，腐蚀=取 min**（二值即"有1/全1"）。
- **开运算** $A\circ B=(A\ominus B)\oplus B$：去毛刺/小物体、平滑轮廓。
- **闭运算** $A\bullet B=(A\oplus B)\ominus B$：填小孔/窄缝。
- 🟡 **击中击不中 (Hit-or-Miss)** $A\circledast B$：形状检测器 = $(A\ominus B_1)\cap(A^c\ominus B_2)$。
- **边界提取**：$\beta(A)=A-(A\ominus B)$（原图减腐蚀）。
- ⚪ **细化/加粗/骨架 (thinning/thickening/skeleton)**。

## 9. 🔴 噪声类型与去噪（IP4，缺，重要概念）

- **加性噪声** 图=原图+噪声 ｜ **乘性噪声** 图=原图×噪声(如激光散斑 speckle)。
- 三种加性：**椒盐(Salt&Pepper，随机黑白点)** / **脉冲(Impulse，随机白点)** / **高斯(Gaussian，正态，最好地建模传感器噪声)**。
- 🔴 **核心对应**：**线性平滑滤波(均值/高斯) → 去高斯噪声**；**非线性(中值) → 去椒盐噪声**。
- **高斯滤波核** $G(x,y)=e^{-(x^2+y^2)/2\sigma^2}$，σ 越大越平滑；均值/高斯都是**低通**会模糊。3×3 均值核=$\frac{1}{9}$全1。
