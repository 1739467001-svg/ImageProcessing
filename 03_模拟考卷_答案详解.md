## Q1　光学　[12]

**(a)** 1/f = 1/u + 1/v = 1/0.3 + 1/0.6 = 3.333 + 1.667 = 5.0 → **f = 0.2 m = 200 mm**。
放大率 m = v/u = 0.6/0.3 = **2** → 像高 = 2 × 0.02 = **0.04 m = 40 mm（倒立）**。

**(b)** 紧贴 → 功率相加：P = 25 + 15 = **40 D**。f = 1/P = 1/40 = 0.025 m = **25 mm**。

**(c)** 透镜功率取决于相对折射率 (n_lens/n_medium − 1)。浸入水中 n_medium：1→1.33，**相对折射率减小 → 透镜变弱、焦距 f 变大、屈光度下降**。原成像面不再合焦，像距变长，需重新对焦或移动屏/传感器。

## Q4　滤波与形态学　[16]

**(a)** 3×3 邻域：
- (2,2) 邻域 {0.2,0.3,0.2, 0.3,0.8,0.3, 0.2,0.3,1.0}：排序中位 = **0.3**；均值 = 3.6/9 = **0.40**。
- (3,3) 邻域 {0.8,0.3,0.2, 0.3,1.0,0.3, 0.2,0.3,0.2}：中位 = **0.3**；均值 = 3.6/9 = **0.40**。
- **效果差异**：中值把噪点 0.8、1.0 直接换成邻域中值 0.3，**有效去除孤立亮噪点且保边**；均值把噪声"摊匀"，结果偏高(0.40)且**边缘被模糊**。

## Q5　卷积与边缘　[10]

第 3 行（中间行），对第 2、3、4 列：
- **Mask A（Prewitt，按讲义的相关算 Gx=右−左，首选）**：`[+3, +3, 0]`；若严格卷积(翻核)则 `[−3, −3, 0]`。
  → 在竖直边(第2、3列附近)产生 ±3 强响应，平坦区(第4列)为 0。**Mask A 检测垂直边缘**（水平方向灰度梯度）。注：此核中间行无加权，严格说是 **Prewitt**（Sobel 中间行为 ±2）。
- **Mask B（Laplacian）**：`[−1, +1, 0]`。→ 边缘两侧出现正负尖峰，平坦区为 0。**Mask B 是二阶导，检测边缘/做锐化，对噪声敏感**。
> 写清你用卷积还是相关；反对称核(Mask A)两者只差符号。

## Q6　矩与 Hough　[14]

**(a)** 点(行i=y,列j=x)：(2,1)(2,2)(2,3)(2,4)(3,2)。x=列：[1,2,3,4,2]，y=行：[2,2,2,2,3]。
- 质心 x̄ = 12/5 = **2.4**，ȳ = 11/5 = **2.2**。
- x́ = [−1.4,−0.4,0.6,1.6,−0.4]，ý = [−0.2,−0.2,−0.2,−0.2,0.8]。
- a = Σx́² = 1.96+0.16+0.36+2.56+0.16 = **5.2**
- c = Σý² = 0.04×4 + 0.64 = **0.8**
- b = 2·Σx́ý = 2·[0.28+0.08−0.12−0.32−0.32] = 2·(−0.4) = **−0.8**
- tan2θ = b/(a−c) = −0.8/(5.2−0.8) = −0.8/4.4 = −0.1818 → 2θ = −10.3° → **θ ≈ −5.2°**。
> 解读：物体几乎水平排列(沿 x 方向伸长)，所以主轴与 x 轴夹角很小（约 −5°），合理。

**(b)** 前三点 (0,1)(1,2)(2,3) 都在直线 **y = x + 1** 上。该线法线 θ=135°，代入 ρ=x cos135+y sin135：
- (0,1)：0+0.707 = **0.707**；(1,2)：−0.707+1.414 = **0.707**；(2,3)：−1.414+2.121 = **0.707** → 三点共享 (ρ=0.707, θ=135°)。
- (3,1)：3(−0.707)+1(0.707) = −2.121+0.707 = **−1.414 ≠ 0.707**。
**∴ 点 (3,1) 不在主线上**（离群点）。

## Q7　深度学习组件　[16]

**(a)** Conv2D = 通道1(逐元素相乘求和) + 通道2 + bias：
- 通道1（核 Sobel，左列−右列）：
  (0.5−0.1)+(0.3−0.4)+(0.2−0.5) = 0.4−0.1−0.3 = **0.0**
- 通道2（核 Laplacian）：4个邻×1 − 中心×4 = (0.4+0.5+0.6+0.4) − 4·0.3 = 1.9 − 1.2 = **0.7**
- 合计 = 0.0 + 0.7 + bias 0.5 = **1.2**；**ReLU(1.2) = 1.2**。

**(b)** 四个 2×2 块：左上{2,1,1,5}、右上{0,3,2,1}、左下{4,0,2,3}、右下{1,6,2,1}。
- **MaxPool** = `[[5, 3], [4, 6]]`
- **AvgPool** = `[[2.25, 1.5], [2.25, 2.5]]`
- **GlobalAvgPool** = 全16数和 34 / 16 = **2.125**

**(c)** 8 个值 {1,3,2,2,4,0,1,3}：μ = 16/8 = **2.0**；σ² = Σ(x−2)²/8 = (1+1+0+0+4+4+1+1)/8 = 12/8 = **1.5**；σ = **1.225**。

## Q8　Viola-Jones　[8]

**(a)** 积分图（II = 本像素 + 上 + 左 − 左上）：
```
 1   3   6   7
 1   4   9  14
 3   7  12  20
 4  11  18  27
```
（右下角 27 = 全图总和，可校验。）

**(b)** Haar = 左两列和(白) − 右两列和(黑)：
- 左两列(c1,c2)和 = II[4][2] = **11**（用积分图直接读）。
- 右两列(c3,c4)和 = 全图 27 − 左两列 11 = **16**。
- **Haar = 11 − 16 = −5**。
- **为什么快**：任意矩形和只需查积分图的**4 个角**（D−B−C+A），是 **O(1)** 整数加减，与矩形大小无关 → Haar 特征可在任意尺度/位置极快算出，这是 Viola-Jones 能在弱硬件上实时的关键。
